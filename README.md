# 🎞 작업 마감 시뮬레이터 (Deadline Manager)

웹툰 및 일러스트 작업을 공정별로 세분화하여 관리할 수 있는 마감 관리 웹앱입니다.

단순한 D-Day 계산을 넘어, 작업의 단계별 진도율을 데이터로 직관적으로 파악할 수 있습니다.

🔗 **[→ 서비스 바로가기](https://gaeun1961.github.io/drawing-work-deadline-manager/)**

---

## ✨ 핵심 기능

### 1. 작업 유형별 맞춤 트래킹

- **🎞 웹툰 모드**: 전체 컷 수를 기준으로 공정별(러프, 선화, 밑색, 명암, 보정) 오늘 작업한 컷 수를 입력하면 구체적인 `%` 진도율이 자동으로 산출됩니다.

- **🎨 일러스트 모드**: 체크박스 방식으로 각 단계(러프, 선화, 밑색, 명암, 보정)의 완료 여부를 간편하게 관리합니다.

### 2. 단계 추가 UX

- **칩 버튼 클릭**: 러프 · 선화 · 밑색 · 명암 · 보정 버튼을 클릭하면 마감일 입력 팝업이 열리고, 날짜를 입력하거나 비워둔 채로 즉시 추가할 수 있습니다. (단계 마감일은 선택사항)

- **✏️ 직접 입력**: 정해진 단계명 외의 공정을 자유롭게 추가할 수 있습니다.

- **작업 중 단계 추가/삭제**: 작업 카드 내에서도 언제든지 단계를 추가하거나 삭제할 수 있습니다.

### 3. 시각화된 대시보드

- **이중 진행률 바 (Dual Progress)**:
  - **평균 진행률**: 모든 공정의 수치를 합산한 실제 작업량의 평균치입니다.
  - **단계 완료율**: 전체 공정 중 최종 완료된 단계가 몇 개인지를 표시합니다.

- **D-Day 상태 알림**: 마감 임박도에 따라 카드 색상이 변경되어 우선순위를 직관적으로 보여줍니다. (Urgent / Warning / OK)

- **단계 접기/펼치기**: 단계 블록 헤더를 클릭하면 상세 내용이 접히고 펼쳐져 많은 단계를 깔끔하게 정리할 수 있습니다.

### 4. 디테일한 관리 도구

- **작업 로그**: 날짜별 작업량과 내용을 기록하여 히스토리를 추적할 수 있습니다.

- **메모 기능**: 클라이언트 요청사항이나 참고 링크 등을 카드별로 기록합니다.

### 5. 사용자 편의성

- **로컬 데이터 저장**: 별도의 가입 없이 브라우저 내에 데이터가 안전하게 저장됩니다.

- **반응형 디자인**: PC와 모바일 환경 어디서든 쾌적하게 사용할 수 있습니다.
