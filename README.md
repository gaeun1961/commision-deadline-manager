# 🎨 커미션 마감 매니저

커미션 작가를 위한 마감 관리 웹앱입니다.
마감일까지 하루에 얼마나 작업해야 하는지 계산해주고, 작업 로그를 기록할 수 있습니다.

🔗 **[→ 바로 사용하기](https://[본인GitHub아이디].github.io/[레포이름])**

---

## ✨ 기능

- 커미션(프로젝트)별 마감일 · 총 작업량 관리
- 단위 선택: 컷(만화) / 장(일러스트)
- 하루 작업량 자동 계산 — 최소 / 안심 / 여유 3단계
- D-Day 카운트다운
- 전체 완료율 바 차트
- 오늘 작업량 입력 및 휴식 기록
- 작업 로그 히스토리
- 데이터 로컬 저장 (새로고침해도 유지)

---

## 🛠 기술 스택

- HTML / CSS / JavaScript (순수 바닐라, 별도 설치 없음)
- 로컬 스토리지로 데이터 저장
